Create a modern Gym Management Chatbot System that works like a real conversational chatbot for websites instead of a simple WhatsApp automation bot. The chatbot should be embeddable on any website and behave like a smart guided assistant with session memory, menu navigation, workflows, reminders, and conversational interactions.

The system should use:

* Frontend Chat Widget (embeddable website chatbot)
* Backend Automation Engine
* PostgreSQL Database
* Admin Panel
* Trainer Panel

The chatbot must support conversational flow management using:

* mode
* pending_step
* pending_data
* updated_at
* bot_mode

The chatbot should maintain user session state and automatically reset inactive sessions after 30 minutes.

The chatbot must support multi-demo or multi-business mode using:

* resto
* gym
* salon
* clinic
* ecommerce

Where:

* bot_mode decides which business chatbot is active
* mode decides current workflow state

Example:

* New visitor arrives
* Chatbot greets user
* User selects Gym Bot
* All future conversations continue inside gym workflow until timeout/reset

Core chatbot behavior:

* Friendly conversational assistant
* Button-based navigation
* Step-by-step forms
* Session memory
* Workflow continuation
* Dynamic menus
* Human-like guided flow without AI dependency
