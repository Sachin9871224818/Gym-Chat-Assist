Update the “Register New Member” modal form and backend integration to fully match our n8n Data Table structure named `members`.

This form must directly send data to the n8n webhook API so that every new member is stored inside the n8n Data Table correctly.

Webhook Endpoint:
POST https://n8n.grindoverdreams.in/webhook/gymbot

Request Method:
POST

Headers:
Content-Type: application/json
x-api-key: skgym2026

Required Form Fields:

1. Full Name → `name`
2. Phone Number → `phone`
3. Age → `age`
4. Gender → `gender`
5. Weight (kg) → `weight`
6. Height (cm) → `height`
7. BMI → auto calculate using:
   BMI = weight / ((height / 100) × (height / 100))
   Save result in `bmi`
8. Goal Dropdown → `goal`
   Options:

   * Weight Loss
   * Muscle Gain
   * Fat Loss
9. Experience Level Dropdown → `experience_level`
   Options:

   * Beginner
   * Intermediate
   * Advanced
10. Injury Notes → `injury_notes`
11. Joining Date → `join_date`
12. Membership Plan Dropdown → `membership_plan`
    Options:

* Monthly
* Quarterly
* Half Yearly
* Yearly

13. Membership Expiry → `membership_expiry`
    Auto-calculate based on selected membership plan:

* Monthly = +1 Month
* Quarterly = +3 Months
* Half Yearly = +6 Months
* Yearly = +12 Months

14. Payment Status Dropdown → `fee_status`
    Options:

* Paid
* Pending

15. Status → `status`
    Default:

* Active

16. Assigned Trainer → `assigned_trainer`
17. Created Time → `created_at`
    Generate automatically in Indian timezone format.
